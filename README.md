Папка содержит 2 математические статьи по теории чесел, а точнее описанее алгоритма, который на основе аддитивных связей Бригад (подмножествпростых чисел - мной введённый термин) 
генерирует всё множество простых (без операции деления!) Свойства этого алгоритма не традиционны, сильно отличаются от имеющихся.
А ещё в папке примеры протоколов моей программы, которая подтверждает своими вычислениями правильность алгоритма.
